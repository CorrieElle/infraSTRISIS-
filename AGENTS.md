# AGENTS.md

Public-safe guardrails for any coding agent editing this repo. This file
intentionally contains only rules already visible on the live site. It is
not the full governance spec, on purpose: this repo is public, and a full
spec would publish the exact things it protects.

Full governance spec lives outside this repo; coding agents must load it
from the local workspace before editing.

## Voice and format (visible on every page)

- No em dashes, ever. Use commas, periods, or line breaks.
- Contractions throughout.
- No prices in copy. No "free," "affordable," "discount." Use "investment," not "cost."
- Avoid corporate filler: "delve," "tapestry," "synergy," "leverage" as a verb where avoidable.

## Design tokens (navy/gold system)

- Base `#0A1A2F`, deep `#071223`, panel `#0E2440`, cards `#12294a`, rules `#1E3A5F`
- Gold `#C8932A` / `#D9AE5F`
- Text `#EEF2F7` / `#B9C4D3` / `#8494A9`
- Type: Playfair Display (headings), DM Sans (body)
- Square buttons; gold-background buttons use navy `#071223` text
